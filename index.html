Projects

The Smart Task Organizer is a productivity application built using Python and SQLite. It allows users to manage daily tasks through a simple command-line or graphical interface. The program categorizes tasks by priority and due date, offering automated reminders using a scheduling library. A key feature is its integration with a natural language parser, allowing users to add tasks using plain English (e.g., “Remind me to submit the report tomorrow at 10 AM”). This project focuses on data structures, file handling, and API integration, helping developers strengthen their understanding of backend logic and user experience design.

The Interactive Web Portfolio is a personal website developed with React.js, HTML5, and CSS3, designed to showcase a programmer’s skills and projects. It features dynamic components, smooth animations, and a contact form that connects to an email API. The portfolio also includes a project gallery powered by a JSON-based backend, enabling easy updates without altering the core code. This project emphasizes front-end development, responsive design, and API connectivity, demonstrating how to create modern, user-friendly web applications that reflect professional coding practices.
